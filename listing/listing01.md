Что выведет программа? Объяснить вывод программы.

```go
package main

import (
    "fmt"
)

func main() {
    a := [5]int{76, 77, 78, 79, 80}
    var b []int = a[1:4]
    fmt.Println(b)
}
```

Ответ:
Выведет слайс b = [77 78 79]. Конструкция a[n:m] позволяет взять слайс от массива или другого слайса от n до m-1
