Что выведет программа? Объяснить вывод программы. Объяснить как работают defer’ы и их порядок вызовов.

```go
package main

import (
	"fmt"
)


func test() (x int) {
	defer func() {
		x++
	}()
	x = 1
	return
}


func anotherTest() int {
	var x int
	defer func() {
		x++
	}()
	x = 1
	return x
}


func main() {
	fmt.Println(test())
	fmt.Println(anotherTest())
}
```

Ответ:
Выведет 2 1, т.к. defer отрабатывает прямо перед выходом из блока функции. В случае с функцией test компилятор свяжет символ x внутри отложенной функции с символом x, обозначающим возвращаемое значение функции. В случае же anotherTest отложенная функция свяжет x с локальной переменной x, и при отработке инкрементирует локальную переменную, а не возвращаемое функцией значение
